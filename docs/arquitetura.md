# Arquitetura

* As funções relacionadas ao gerenciamento das casas do jogo
da velha ficarão no módulo **jogovelha.py**.

* o estado de cada casa do jogo será representada por uma string: "."
para cada vazia; "x" para casa ocupada pelo 1º jogador; 
"O" para casa ocupada pelo 2º jogador

* A função iniciaizar () retornará uma lista 3x3, onde cada posição conterá uma string para indicar o
estado de uma casa do jogo. A função retornará todas as casas inicialmente 
vazias.
